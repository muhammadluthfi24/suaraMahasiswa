INSTALASI
=======================
1. Instal XAMPP atau WAMPP
2. Buka panel Kontrol XAMPP dan mulai [apache] dan [mysql]
3. ekstrak file di website C:\xampp\htdocs
4. buka tautan localhost/phpmyadmin
5. Import file database dan diberi nama Suara_Mahasiswa1, file database berada didalam folder yang bernama database dengan nama file pengaduan_mahasiswa.sql
6. setelah import database, buka browser apa saja dan ketik http://localhost/Suara_Mahasiswa1

PENGGUNAAN USER MAHASISWA
=======================
1. Login dengan user mahasiswa
2. Mahasiswa dapat melakukan pengaduan pada bar navigasi "Tambah Pengaduan" dengan menuliskan laporan dan menambah bukti dengan menginput gambar pada form tersebut
3. Pada bar navigasi "Daftar Laporan" mahasiswa dapat melihat riwayat laporan pengaduan yang telah selesai diadukan dan sudah di tanggapi oleh petugas
=======================

PENGGUNAAN USER PETUGAS
=======================
1. Login dengan akun petugas
2. Pada bar navigasi terdapat "Dashboard" yang berfungsi untuk melihat Laporan status proses dan laporan status diterima
3. Kemudian pada bar navigasi "Pengaduan" petugas dapat melihat daftar pengaduan yang telah dikirim oleh mahasiwa dan petugas dapat menanggapi pengaduan tersebut dengan klik tombol MORE atau juga dapat menghapus pengaduan tersebut
4. Di bar navigasi "Respon" petugas dapat melihat data pengaduan mahasiswa yang telah direpon
=======================

PENGGUNAAN USER ADMIN
=======================
1. Login dengan akun admin
2. Pada bar navigasi terdapat "Dashboard" yang berfungsi untuk melihat Laporan status masuk dan laporan status selesai
3. Didalam bar navigasi "Registrasi" terdapat data akun mahasiswa dan dapat menambah/edit/hapus data mahasiswa
4. Pada bar navigasi "Pengaduan" berisi data pengaduan yang telah selesai diproses dan dapat menanggapi pengaduan yang sedang diproses
5. Pada bar navigasi "Respon" menampilkan data pengaduan yang telah selesai diproses
6. Didalam bar navigasi "User" terdapat data akun admin/petugas dan dapat menambah/edit/hapus data admin/petugas
7. Bar navigasi "Laporan" berisi data-data pelapor, nama petugas yang menanggapi, waktu pengaduan.
